# Code of Conduct

Keep collaboration direct, respectful, and focused on the work.

Expected behavior:

- Be clear and specific.
- Discuss technical tradeoffs in good faith.
- Keep criticism about the code or design, not the person.
- Respect API terms, data licenses, and source access rules.

Unacceptable behavior:

- Harassment or personal attacks
- Sharing secrets or private credentials
- Encouraging market manipulation or illegal trading behavior
- Misrepresenting research output as guaranteed financial advice

Issues can be handled by the repository owner.
